Decoder-only transformer architecture trained on Shakespeare's works.

## Training
```bash
python train.py 
```

## Data
The `logs/` directory contains the training logs.
The more.txt file contains 10000 tokens generated by the trained model.

## Hyperparameters

These hyperparameters were used for training the model:

| Parameter | Value |
|---|---|
| `batch_size` | 64 |
| `block_size` | 256 |
| `max_iters` | 5000 |
| `eval_interval` | 500 |
| `learning_rate` | 3e-4 |
| `device` | `"cuda"` if available, else `"cpu"` |
| `eval_iters` | 200 |
| `n_embd` | 384 |
| `n_heads` | 6 |
| `n_blocks` | 6 |
| `dropout` | 0.2 |
